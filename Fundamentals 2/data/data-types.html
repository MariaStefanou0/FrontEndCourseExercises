Data types

Type What are they? Example
String Variable with a series of characters 
  let lastName = "Johnson";
Number Variable made with numbers 
  let age = 30;
Boolean True || False 
  age = lastName; // False
Symbol A “symbol” represents a unique identifier. A value of this type can be created using Symbol(). Symbols are guaranteed to be unique. 
  let id = Symbol(); 
  let id = Symbol("id"); 
  let id1 = Symbol("id");
  let id2 = Symbol("id");
                
  alert(id1 == id2); // false
BigInt is a built-in object that provides a way to represent whole numbers larger than 253 - 1 
  let previouslyMaxSafeInteger = 9007199254740991n

  let alsoHuge = BigInt(9007199254740991)
  // ↪ 9007199254740991n
Undefined Variable without a value 
  let birth;

  birth = undefined;
Null Null is "nothing" 
  let person = {home :"Athens", eyes : "brown"};
  
  person = null;
This types ABOVE are considered as Primitives. They are immutable, they cannot be altered 
  let name = 'Flavio'
  let secondName = name
  name = 'Roger'
  secondName //'Flavio'
